첫번째 타이틀

페이지 상단에 추가되며 대부분의 페이지에서 생략되며 아래와 같은 속성을 가진다.
모든 타이틀은 line-height 의 여백을 포함하여 margin 이 입력되어 있음

  • margin: 0 72px
  • font-size: 60px
  • color: #000
  • line-height: 78px
  • font-weight: 500
두번째 타이틀 두번째

주로 사용되는 타이틀로 아래와 같은 속성을 가진다.

  • margin: 150px 72px 42px
  • font-size: 54px
  • line-height: 72px
  • color: #000
  • font-weight: 500
  • 페이지의 첫번째 요소로 올때는 margin-top: 0
세번째 타이틀

  • font-size: 54px
  • line-height: 66px
  • color: #000
  • font-weight: 500
  • 페이지의 첫번째 요소로 올때는 margin-top: 0
